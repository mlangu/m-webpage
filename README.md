# M Language — Website

This repository contains the official website for the M programming language.

The site presents the language, its philosophy, documentation, and instructions for downloading and running the interpreter.

---

## Overview

The website is designed to reflect the core ideas of M:

* simplicity
* clarity
* minimalism

It avoids unnecessary complexity and focuses on readability, both in design and content.

---

## Structure

The site is composed of simple static pages:

* `index.html` — main page
* `download.html` — installation and usage
* additional pages for documentation and examples

All styling is embedded directly in the HTML to keep the project lightweight and easy to understand.

---

## Design

The visual style follows a minimal and slightly retro aesthetic:

* light paper-like background
* dark text
* monospace headings (`Courier Prime`)
* simple layout with no frameworks

The goal is to match the philosophy of the language itself.

---

## Running Locally

No build tools are required.

Simply open the files in your browser:

```bash
xdg-open index.html
```

or

```bash
open index.html
```

---

## Editing

All pages are plain HTML and CSS.

You can modify:

* text content directly in the files
* styles inside the `<style>` block
* structure using standard HTML

No external dependencies are required.
