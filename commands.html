M Language: Guia de Comandos

Referência rápida de sintaxe e uso para a versão 0.1

Variáveis

Cria e armazena valores na memória. Não é necessário definir o tipo.

nome = "João"
idade = 25
lista = []
objeto = {}

Entrada e Saída

Interaja com o usuário através do terminal.

// Saída automática
"Olá Mundo"

// Capturar entrada
resposta = ask "Deseja sair?"

Listas

Coleções de dados indexadas começando em 0.

frutas = ["Maçã", "Uva"]

// Adicionar item
add "Banana" to frutas

// Obter tamanho
total = size frutas

// Acesso por índice
item = frutas[0]

Objetos

Estruturas chave-valor para organizar informações.

carro = {
    marca = "Ford"
    ano = 2022
}

// Acesso por ponto
"Marca: " + carro.marca

Matemática

Operações aritméticas com precedência padrão.

soma = 10 + 5
sub  = 20 - 3
mult = 5 * 4
div  = 100 / 2

// Concatenação de texto
msg = "Idade: " + 25

Lógica

Comparações que retornam verdadeiro (1) ou falso (0).

// Operadores
==  (Igual)
>   (Maior)
<   (Menor)
>=  (Maior ou igual)
<=  (Menor ou igual)

Fluxo de Controle (Condicionais e Loops)

Controle o caminho que o seu código deve seguir. Todo bloco termina com end.

Condicional IF

if idade >= 18:
    "Maior de idade"
else:
    "Menor de idade"
end

Loops (Repeat e While)

// Repetição controlada
count = 0
while count < 5:
    count = count + 1
end

// Repetição infinita
repeat:
    a = ask "Parar? (y/n)"
    if a == "y":
        stop
    end
end

Funções

Defina blocos de código para serem reutilizados.

func saudar:
    "Seja bem-vindo à M Language!"
end

// Chamar função
saudar
