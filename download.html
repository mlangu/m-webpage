Obter a M Language

Download via GitHub
AVISO: A M Language está em fase experimental (v0.1). O sistema exige interação via terminal e compilação manual. Erros e instabilidades são esperados nesta versão.

1. Pré-requisitos

Você precisará de um ambiente Unix (Linux ou macOS) com os seguintes pacotes instalados:

  • gcc (Compilador C)
  • make (Gerenciador de compilação)

2. Obter o código

Você pode clonar o repositório diretamente:

$ git clone https://github.com/mlangu/m-language.git
$ cd m-language

3. Compilação

Execute o comando abaixo para construir o interpretador:

$ make --quiet

Isso gerará um executável chamado m.

4. Seu primeiro código

Crie um arquivo chamado ola.m:

"Olá da linguagem M!"

Execute:

$ ./m ola.m

Por que no terminal?

M foi projetada para ser simples e direta. O uso do terminal permite interação com o interpretador sem camadas adicionais, mantendo a linguagem leve e fiel à sua proposta minimalista.
